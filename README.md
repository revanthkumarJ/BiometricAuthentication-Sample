# BiometricAuthenticator-Sample

A simple Android sample app demonstrating **Biometric Authentication** using **Jetpack Compose** and **AndroidX Biometric** APIs.

## Features
- Shows a biometric prompt with title and description.
- Handles success, error, and cancellation states.
- Redirects to biometric enrollment settings if not set up.
- Displays authentication result on screen.

## Tech Stack
- Kotlin  
- Jetpack Compose  
- AndroidX Biometric  
- Material 3  

## Files
- **MainActivity.kt** – Contains the UI and integrates biometric authentication.  
- **BioMetricPromptManager.kt** – Manages the biometric prompt and authentication results using Kotlin Flows.
